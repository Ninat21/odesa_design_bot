> Интеллектуальная платформа управления профессиональными сообществами.

---

# О проекте

Odesa Community Platform — это система управления профессиональным сообществом нового поколения.

Проект объединяет CRM участников, Telegram, мероприятия, аналитику, искусственный интеллект и базу знаний в единую платформу.

Первоначально система разрабатывается для сообщества дизайнеров Одессы, однако архитектура проектируется как универсальная платформа для любых профессиональных сообществ.

# Основные принципы

## Единый источник истины

PostgreSQL является единственным источником данных.

Все сервисы работают только через базу данных.

---

## Разделение ответственности

Каждый модуль отвечает только за одну область.

Например:

- Telegram не знает про AI.
- AI не изменяет фактические данные.
- Analytics не изменяет сообщения.

---

## История важнее текущего состояния

Любое изменение может быть восстановлено.

Ничего не удаляется без необходимости.

---

## AI не заменяет факты

Все выводы AI хранятся отдельно.

Фактические данные никогда не перезаписываются AI.

---

## Масштабируемость

Архитектура должна поддерживать:

- несколько сообществ
- несколько Telegram-групп
- веб-интерфейс
- мобильное приложение
- несколько AI-моделей

без изменения существующей структуры данных.
