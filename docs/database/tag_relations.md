# Таблиця `tag_relations`

## Призначення

Таблиця реалізує зв'язок між тегами та будь-якими сутностями системи.

---

# Поля

| Поле | Тип | NULL | Опис |
|------|------|------|------|
| id | BIGSERIAL | ❌ | Первинний ключ |
| tag_id | BIGINT | ❌ | tags.id |
| entity_type | VARCHAR(50) | ❌ | Тип сутності |
| entity_id | BIGINT | ❌ | ID сутності |
| created_at | TIMESTAMP | ❌ | Створення |

---

# Приклади

```
tag → user

tag → event

tag → message

tag → knowledge_article
```

---

# Причини такої структури

Одна таблиця дозволяє використовувати універсальну систему тегів у всій платформі.
