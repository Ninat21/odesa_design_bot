# Таблиця `settings`

## Призначення

Таблиця містить глобальні налаштування платформи.

---

# Поля

| Поле | Тип | NULL | Опис |
|------|------|------|------|
| id | BIGSERIAL | ❌ | Первинний ключ |
| key | VARCHAR(100) | ❌ | Назва параметра |
| value | TEXT | ✅ | Значення |
| description | TEXT | ✅ | Опис |
| updated_at | TIMESTAMP | ❌ | Остання зміна |

---

# Обмеження

```
UNIQUE(key)
```

---

# Причини такої структури

Дозволяє змінювати конфігурацію без зміни коду.
