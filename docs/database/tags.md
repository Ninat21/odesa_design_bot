# Таблиця `tags`

## Призначення

Таблиця містить єдиний довідник тегів, які використовуються в системі.

Теги можуть застосовуватися до різних сутностей:

- користувачів;
- статей;
- повідомлень;
- подій.

---

# Поля

| Поле | Тип | NULL | Опис |
|------|------|------|------|
| id | BIGSERIAL | ❌ | Первинний ключ |
| name | VARCHAR(100) | ❌ | Назва тегу |
| color | VARCHAR(20) | ✅ | Колір |
| description | TEXT | ✅ | Опис |
| created_at | TIMESTAMP | ❌ | Створення |
| updated_at | TIMESTAMP | ❌ | Оновлення |

---

# Обмеження

```
UNIQUE(name)
```

---

# Причини такої структури

Єдиний довідник тегів дозволяє використовувати однакові теги в різних частинах системи без дублювання.
