# Таблиця `knowledge_sources`

## Призначення

Таблиця пов'язує статті бази знань із джерелами, на основі яких вони були створені.

Одне джерело може використовуватися в декількох статтях.

Одна стаття може мати необмежену кількість джерел.

---

# Відповідальність

Таблиця реалізує зв'язок між статтею та її джерелами.

---

# Поля

| Поле | Тип | NULL | Опис |
|------|------|------|------|
| id | BIGSERIAL | ❌ | Первинний ключ |
| knowledge_article_id | BIGINT | ❌ | knowledge_articles.id |
| source_type | VARCHAR(30) | ❌ | Тип джерела |
| source_id | BIGINT | ❌ | ID джерела |
| created_at | TIMESTAMP | ❌ | Дата створення |

---

# Типи джерел

- message
- saved_message
- article
- event

---

# Зовнішні ключі

```
knowledge_article_id → knowledge_articles.id
```

---

# Індекси

```
knowledge_article_id

source_type

source_id
```

---

# Причини такої структури

Дозволяє завжди визначити, на основі яких матеріалів була сформована стаття.

Це особливо важливо для матеріалів, створених AI.
