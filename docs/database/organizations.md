# Таблиця `organizations`

## Призначення

Таблиця містить довідник організацій, компаній, студій, навчальних закладів та інших місць, з якими можуть бути пов'язані учасники спільноти.

Одна організація може бути пов'язана з багатьма користувачами.

---

# Відповідальність

Таблиця відповідає лише за інформацію про організацію.

Не містить інформації про конкретних користувачів.

---

# Поля

| Поле | Тип | NULL | Опис |
|------|-----|------|------|
| id | BIGSERIAL | ❌ | Первинний ключ |
| name | VARCHAR(255) | ❌ | Назва організації |
| legal_name | VARCHAR(255) | ✅ | Повна юридична назва |
| website | VARCHAR(255) | ✅ | Офіційний сайт |
| linkedin | VARCHAR(255) | ✅ | LinkedIn компанії |
| instagram | VARCHAR(255) | ✅ | Instagram |
| city | VARCHAR(255) | ✅ | Місто |
| country | VARCHAR(255) | ✅ | Країна |
| description | TEXT | ✅ | Опис |
| created_at | TIMESTAMP | ❌ | Створення запису |
| updated_at | TIMESTAMP | ❌ | Останнє оновлення |

# Правила оновлення

Інформація про організацію може змінюватися адміністратором.

---

# Джерела даних

- Адміністратор
- Імпорт
- Website
- Luma

---

# Причини такої структури

Організація існує незалежно від користувачів.

Це дозволяє уникнути дублювання однакових назв компаній у профілях.

---

# Майбутній розвиток

Можливе додавання:

- логотипу;
- галузі;
- кількості співробітників;
- типу організації;
- країни реєстрації.
