# Таблиця `events`

## Призначення

Таблиця `events` містить інформацію про всі заходи спільноти.

Подія може бути створена вручну або імпортована із зовнішнього сервісу (наприклад, Luma).

Таблиця є центральною сутністю модуля Events.

---

# Відповідальність

Таблиця відповідає за:

- загальну інформацію про подію;
- місце проведення;
- дату та час;
- статус;
- зовнішні ідентифікатори;
- організаційну інформацію.

Не містить інформації про учасників або відгуки.

---

| organizer_name | VARCHAR(255) | ✅ | Організатор |
| venue_name | VARCHAR(255) | ✅ | Назва місця |
| address | TEXT | ✅ | Адреса |
| city | VARCHAR(255) | ✅ | Місто |

# Причини такої структури

Подія повинна існувати незалежно від учасників.

Це дозволяє:

- імпортувати події до появи реєстрацій;
- вести календар;
- аналізувати статистику;
- повторно використовувати інформацію.

---

# Майбутній розвиток

- кількох локацій;
