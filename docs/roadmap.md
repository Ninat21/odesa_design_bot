# Roadmap

## Призначення

Roadmap визначає довгостроковий план розвитку платформи.

Документ описує етапи розвитку, а не конкретні задачі програмування.

---

# Поточний статус

Поточна версія:

**v0.1 — Архітектура**

На цьому етапі виконується:

- проєктування архітектури;
- проєктування бази даних;
- створення Telegram Bot;
- налаштування PostgreSQL;
- підготовка AI-модуля.

# Правила розвитку

Перед додаванням нового функціоналу необхідно відповісти на питання:

- Чи вирішує ця функція реальну проблему?
- Чи не дублює вона вже існуючу?
- Чи відповідає вона архітектурі?
- Чи буде вона корисною через рік?

Лише після цього функція може бути включена до Roadmap.
